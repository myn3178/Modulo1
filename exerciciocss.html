Metadados do documento

Elemento Descrição base O elemento HTML Base ( base ) especifica o endereço (URL) utilizado por todos os endereços relacionados contidos dentro de um documento. Há um número máximo de 1 (um) elemento Base base do documento. head O elemento HTML headfornece informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos. link O Elemento HTML link especifica as relações entre o documento atual e um recurso externo. Possíveis usos para este elemento incluem a definição de uma estrutura relacional para navegação. Este elemento é mais usado para vincular as folhas de estilo. meta O elemento **HTML meta**define qualquer informação de metadados que não pode ser definida por outros elementos HTML. ( base, link, script, styleou title). style O elemento H T M L style contém informações de estilo para um documento ou uma parte do documento. As informações de estilo específicas estão contidas neste elemento, geralmente no CSS . title>O elemento H T M L title ( Elemento h t m l Título ) define o título do documento, exibido na barra de título de um navegador ou na aba da página. Pode conter somente texto e quaisquer marcações contidas no texto não são interpretadas.
